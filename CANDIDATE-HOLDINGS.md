# CANDIDATE-HOLDINGS.md

## 目的
建立 `stock-learning-site` 的第一版候选持仓池，把前面的资产 / 行业 / 个股研究真正接到组合层。

说明：
- 当前为候选池，不等于实盘持仓
- 先做结构化观察池，后续再结合市场环境动态调整

---

## 一、候选池分层

### A. 核心配置池
适合承担较高关注度和中长期跟踪权重。

#### 1. china-shenhua
- 资产：A股
- 主线：高股息 / 能源 / 现金流稳定
- 角色：**防守型核心候选**
- 适合环境：
  - 风险偏好偏弱
  - 市场偏好确定性现金流
  - 利率回落或稳定
- 关键观察点：
  - 分红持续性
  - 煤价中枢
  - 资本开支纪律

#### 2. haitian
- 资产：A股
- 主线：品牌消费 / 调味品龙头 / 高质量消费
- 角色：**核心资产候选**
- 适合环境：
  - 内需修复
  - 核心资产重估
  - 消费预期改善
- 关键观察点：
  - 动销恢复
  - 渠道库存
  - 毛利率与净利率稳定性

### B. 观察池
适合继续跟踪，等待行业/估值/宏观条件进一步确认。

#### 1. cr-mixc
- 资产：港股
- 主线：物业管理 / 消费服务 / 低估值修复
- 角色：**修复型观察候选**
- 适合环境：
  - 地产政策边际改善
  - 港股风险偏好回暖
  - 中国资产估值修复
- 关键观察点：
  - 地产链风险传导
  - 经营独立性
  - 估值修复与基本面匹配度

#### 2. semiconductor 相关标的
- 资产：A股 / 港股 / 美股（扩展位）
- 主线：科技成长 / 国产替代 / 周期复苏
- 角色：**高弹性观察池**
- 适合环境：
  - 利率回落
  - 风险偏好提升
  - 库存周期见底
- 关键观察点：
  - 下游需求恢复
  - 库存去化
  - 估值修复是否快于盈利修复

#### 3. innovative-drugs 相关标的
- 资产：A股 / 港股
- 主线：创新药 / 医药成长 / 政策预期改善
- 角色：**成长观察池**
- 适合环境：
  - 风险偏好修复
  - 政策预期改善
  - 成长风格回暖
- 关键观察点：
  - 管线兑现
  - 政策变化
  - 估值消化程度

## 二、动态调整规则
### 新增进入候选池
满足至少一类条件：
- 宏观顺风开始形成
- 行业景气开始改善
- 个股估值进入合理区间
- 风险收益比明显改善

### 移出候选池
满足至少一类条件：
- 原有逻辑被破坏
- 赔率显著下降
- 宏观环境明显逆风
- 行业或公司风险上升
