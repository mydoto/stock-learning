# global-asset-framework.md

## 目标
把现有 `stock-learning-site` 的研究体系，从“股票学习系统”升级为“全球资产配置研究系统”。

核心闭环：
**宏观变量 → 大类资产判断 → 行业映射 → 个股观察 → 组合配置 → 输出复盘 → 提醒执行**

---

## 一、研究总框架

### 1. 宏观变量层
这是整个系统的最上游。

重点跟踪 6 类变量：
- **利率**：无风险利率、实际利率、长短端利差
- **通胀**：CPI、PPI、核心通胀、通胀预期
- **增长**：GDP、PMI、社融、就业、消费、投资
- **流动性**：货币政策、财政政策、信用扩张、市场流动性
- **汇率**：美元指数、人民币汇率、主要货币走势
- **风险偏好**：波动率、信用利差、估值水平、市场风格

这些变量的作用不是单独记录，而是回答：
- 当前宏观环境属于什么阶段？
- 哪类资产顺风？哪类资产逆风？
- 哪些行业、风格、公司更受益？

---

## 二、大类资产层

### 资产分类
- **权益**：A股 / 港股 / 美股
- **固收**：国债 / 美债 / 信用债
- **商品**：黄金 / 原油 / 铜
- **外汇**：美元指数 / 人民币汇率
- **现金**：货币基金 / 短债 / 无风险利率
- **另类**：REITs / Crypto（可选）

### 研究目标
每类资产都要回答：
1. 这类资产由什么驱动？
2. 当前最关键的影响变量是什么？
3. 当前环境下偏多、偏空还是中性？
4. 在组合中扮演什么角色？
   - 进攻
   - 防守
   - 对冲
   - 现金替代
5. 需要盯哪些高频指标？

---

## 三、权益映射层

权益不是孤立研究，而是从上层资产视角往下拆。

### 研究路径
1. 先判断权益整体环境：
   - 风险偏好是否上升？
   - 利率环境是否友好？
   - 盈利周期是否改善？
2. 再看市场分化：
   - A股、港股、美股哪个更优？
   - 成长、价值、高股息哪个更优？
3. 再下钻到行业：
   - 哪些行业与宏观变量匹配度高？
4. 最后落到个股：
   - 哪些公司最能体现这条主线？

### 行业与个股的定位
- **行业研究**：权益映射层
- **个股研究**：标的观察层

也就是说，现有的行业和个股内容不是重做，而是放进这个结构里继续用。

---

## 四、组合配置层

### 组合配置需要回答 4 个问题
1. **配什么**：当前主配资产有哪些？
2. **为什么配**：依据哪些变量和逻辑？
3. **配多少**：权重如何设定？
4. **什么时候调**：哪些条件触发再平衡？

### 配置分层
- **战略配置**：长期框架，不频繁改
- **战术配置**：阶段性超配/低配
- **风险预算**：控制单一资产、主题、标的暴露
- **仓位管理**：控制回撤与节奏

### 再平衡逻辑
至少定义：
- 估值偏离阈值
- 宏观变量拐点
- 风险预算超限
- 组合回撤阈值

---

## 五、输出层

系统要稳定产出 3 类输出：

### 1. 每日市场观察
关注：
- 隔夜市场
- 利率 / 美元 / 黄金 / 原油
- 重点行业与个股异动
- 当日 3 条核心观察

### 2. 每周配置复盘
关注：
- 本周宏观变量变化
- 大类资产表现
- 行业与个股主线
- 当前配置是否需要调整
- 下周观察重点

### 3. 专题研究输出
围绕：
- 单一知识主题
- 单一资产
- 单一行业
- 单一公司
