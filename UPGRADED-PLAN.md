# UPGRADED-PLAN.md

## 计划定位
本计划不是推倒重来，而是在 `PLAN.md`、`REMINDER-PLAN.md`、`REMINDER-TODO.md` 以及现有 data / content 基础上，做一次**融合式升级**。

原则：
1. **保留已完成工作**：知识卡片、行业研究、个股研究、输出内容、数据层、提醒设计都继续沿用。
2. **补齐缺失层**：在原有“股票学习系统”上，补上“全球资产配置”和“组合配置”两层。
3. **保持低摩擦执行**：不大改目录，不打碎原文件；优先新增少量关键文件和模板。
4. **先打通闭环，再扩内容量**：先保证“学习→研究→输出→复盘→提醒”能连续运转。

### 目标 1：保留现有股票研究底座

## 三、融合方案：原内容如何接入升级版

### 1. 知识卡片层：继续保留，升级为“底层原理库”
原有知识点如：
- 利率
- ROE
- ROIC
- 自由现金流
- 商业模式
- 行业景气度
- 仓位管理

这些不要重写，直接升级定位：
- 原来：股票研究知识点
- 现在：资产配置与股票研究共用的底层原理库

#### 接入方式
每张知识卡片新增两个字段（哪怕先在文档中手工维护也行）：
- **影响资产**：股票 / 债券 / 黄金 / 汇率 / 商品
- **应用场景**：估值判断 / 周期判断 / 配置判断 / 风险控制

### 2. 行业研究层：继续保留，升级为“权益映射层”
原有行业研究不用动结构，只增加两类视角：
- 该行业受哪些宏观变量影响最大
- 它在什么资产环境下更占优/更吃亏

#### 举例
- 半导体：利率敏感、成长风格敏感、美元周期敏感
- 白酒：内需、消费、风险偏好、估值溢价
- 高股息：利率比较、现金流稳定性、避险环境

### 3. 个股研究层：继续保留，升级为“资产配置下的标的观察层”
原有个股研究保留，但以后每个个股要补充：
- 属于哪个行业主线
- 受哪些宏观变量影响
- 更适合长期配置 / 观察名单 / 周期波段 / 防守配置

### 4. 输出层：继续保留，升级为“配置观点输出层”
原输出不作废，但未来新增 3 类输出：
- 每日市场观察
- 每周配置复盘
- 主题资产研究

### 5. 提醒层：沿用原 reminder 设计，不重做
`REMINDER-PLAN.md` 和 `REMINDER-TODO.md` 保留。
升级动作不是改逻辑，而是把提醒内容和新框架绑定：
- 08:30：市场扫描（宏观 + 大类资产）
- 12:30：知识学习（底层原理/大类资产）
- 21:30：晚间复盘（行业/个股/组合）
- 周日 20:30：周度配置复盘
