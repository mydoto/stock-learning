# 行业景气度分析框架

## 核心逻辑

**景气度 = 供需关系 + 价格信号 + 库存周期**

研究景气度，本质是在问：
**这个行业现在的日子是在变好还是变差，变化的速度是快还是慢？**

## 四个维度

### 1. 需求
- 终端需求有没有扩张？
- 增量需求来自哪里（新市场 / 存量替换 / 政策刺激）？

### 2. 供给
- 行业产能是否过剩？
- 新增产能节奏如何？
- 头部格局有没有变化？

### 3. 价格
- 产品价格或服务单价在涨还是跌？
- 涨价是景气上行信号，跌价是景气下行信号。

### 4. 库存
- 上游 / 渠道 / 终端的库存水位如何？
- 去库存阶段价格承压，补库存阶段价格有支撑。

## 研究应用

- 判断当前行业处于哪个景气阶段（上行 / 顶部 / 下行 / 底部）
- 把景气变化与股价预期做对比，找"预期差"
- 景气领先指标往往是订单、价格、库存，先于财报反映
