# 权益总览研究框架

- 名称：权益
- 分类：权益
- 子类：A股 / 港股 / 美股
- 市场：中国 / 美国 / 全球

---

## 一、资产定义
权益资产的核心收益来源是：
- 企业盈利增长
- 估值扩张/收缩
- 分红回报

权益是组合中最重要的长期增长来源，但同时波动也更高，对宏观环境和风险偏好更敏感。

## 二、核心驱动因子
权益最关键的三个驱动因子：
1. **盈利周期**
2. **利率水平**
3. **风险偏好/流动性**

补充驱动：
- 政策预期
- 行业景气度
- 汇率与外资流动

## 三、定价/估值框架
权益估值通常看：
- PE / PB / PS / EV/EBITDA
- 利润增速与预期变化
- 利率与风险溢价

简单理解：
- 利率下降 + 盈利改善 → 权益通常顺风
- 利率上升 + 估值过高 → 高估值成长更容易受压
- 风险偏好提升 → 高 Beta / 成长 / 小盘更活跃

## 四、关键跟踪指标
### 高频指标
- 主要股指表现（A股 / 港股 / 美股）
- 成交额 / 换手率 / 风格指数
- 北向/南向资金、ETF 资金流向
- 美债收益率与美元变化

### 中频指标
- 企业盈利预期
- PMI / 社融 / 消费 / 投资数据
- 政策变化
- 行业景气度数据

## 五、当前市场环境判断
权益更占优的环境：
- 利率平稳或下行
- 盈利周期改善
- 流动性宽松
- 风险偏好回暖

权益更承压的环境：
- 利率快速抬升
- 盈利下修
- 风险偏好下降
- 外部冲击显著上升

## 六、风险点
- 盈利不及预期
- 估值过高后回撤
- 利率超预期上行
- 汇率与外资流向恶化

## 七、配置角色
权益在组合中的角色：
- **长期增长核心**
- **获取超额收益的主战场**

但权益内部还要区分：
- 成长
- 价值
- 高股息
- 周期
- 防守

## 八、与其他模块的映射
- 知识卡片：利率、ROE、ROIC、现金流、景气度、商业模式
- 行业研究：直接作为权益映射层
- 个股研究：直接作为标的观察层
- 债券 / 黄金 / 汇率：影响权益估值和风格表现

## 九、结论
### 当前判断
权益仍是长期收益最核心资产，但需要放在宏观、利率、盈利、风险偏好的框架下理解，而不是单独看 K 线。

### 观察清单
- 利率方向
- 盈利预期变化
- 风险偏好
- 行业主线变化
