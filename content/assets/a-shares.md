# A股研究框架

- 名称：A股
- 分类：权益
- 子类：沪深主板 / 创业板 / 科创板
- 市场：中国

---

## 一、资产定义
A股是中国本土权益市场，核心驱动力来自：
- 国内经济增长预期
- 政策方向
- 流动性环境
- 企业盈利周期
- 市场风险偏好

A股的一个重要特点是：
- 政策影响较大
- 风格切换明显
- 行业轮动较快
- 散户与机构共存，情绪波动较大

## 二、核心驱动因子
A股最关键的三个驱动因子：
1. **国内经济预期**
2. **流动性与政策环境**
3. **风险偏好与风格轮动**

补充驱动：
- 社融与信用扩张
- 房地产周期
- 财政刺激方向
- 北向资金流入流出

## 三、定价/估值框架
A股估值常用：
- PE / PB
- PEG（成长股）
- 股息率（高股息）
- 行业景气度与盈利修正

简单理解：
- 宽信用 + 流动性宽松 → A股整体更顺风
- 利率下行 + 成长预期改善 → 成长板块更占优
- 经济复苏 + 顺周期预期增强 → 周期与价值更占优

## 四、关键跟踪指标
### 高频指标
- 上证指数 / 沪深300 / 中证1000 / 创业板指
- 两市成交额
- 北向资金
- 风格指数（成长 / 价值 / 高股息）

### 中频指标
- 社融 / M1 / M2
- PMI
- 房地产与消费数据
- 政策会议与财政发力方向
- 上市公司盈利预期

## 五、当前市场环境判断
A股更占优的环境：
- 政策友好
- 流动性边际改善
- 盈利预期修复
- 风险偏好回升

A股更承压的环境：
- 经济预期转弱
- 政策低于预期
- 风险偏好快速下行
- 外资持续流出

## 六、风险点
- 政策力度不及预期
- 经济修复弱于预期
- 风格过度拥挤后的回撤
- 外部环境冲击导致风险偏好下降

## 七、配置角色
A股在组合中的角色：
- **中国权益核心配置**
- **政策驱动和产业机会的重要来源**
- **获取超额收益的主战场之一**

## 八、与其他模块的映射
- 宏观变量：社融、利率、财政政策、风险偏好
- 行业映射：消费、科技、高股息、周期、制造业升级
- 个股映射：现有个股研究可直接接入

## 九、结论
### 当前判断
A股更适合放在“政策 + 流动性 + 盈利预期”的三维框架里理解，而不是只看指数涨跌。

### 观察清单
- 政策边际变化
- 社融与信用扩张
- 北向资金方向
